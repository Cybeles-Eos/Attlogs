# Attlogs

Follow these steps to get the application up and running locally:

1. Clone the project repository

   ```sh
   git clone https://github.com/Cybeles-Eos/Attlogs.git
   cd attlogs
   ```

2. Run This Command In the Terminal

   ```sh
    git checkout -b <your-initials-plus-lastname>     #Ex. "dijcruz" "dbbarcelon"
    git pull origin main
    git pull origin staging
   ```

3. Create Database in phpadmin
    
    ```
    - Open Xammp Create new database name "attlogs"
    - Import Our Database SQL File inside database folder 
    ```
